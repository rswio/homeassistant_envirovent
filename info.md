# EnviroVent ATMOS

Local Home Assistant control for an **EnviroVent ATMOS PIV** (ATL‑A) unit — fan
speed & variable airflow, boost, auto heater, summer mode, filter counter and status
sensors — all over your LAN, no cloud.

**Unofficial & reverse‑engineered — use at your own risk.** Installer/commissioning
settings are intentionally not exposed. The unit accepts only one connection at a
time, so close the myenvirovent phone app when Home Assistant should have it.

Add via **Settings → Devices & Services → Add Integration → EnviroVent ATMOS** and
enter the unit's IP (port 1337). See the README for full details.
